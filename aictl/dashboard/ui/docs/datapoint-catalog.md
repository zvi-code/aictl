## Legend

- **RAW**: Direct reading from a DB table or config lookup
- **DEDUCED**: Requires calculation over queried data
- **AGGREGATED**: SUM/COUNT/GROUP BY over queried rows
- **\***: Dynamic — live provenance JSON updated each cycle (`GET /api/datapoints?key=<key>`)
- `?param?` in queries: bind parameter (tool name, PID, timestamp, session ID)
